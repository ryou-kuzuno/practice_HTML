#paddingとmarginの違い

```
paddingは要素を含んだ風船の中のようなもので、指定してやれば風船の大きさを変えてその中にある要素の位置を変えることができる。

例　しぼんでいて、要素とくっついている(padding: 0;)　
    膨らんでいて要素と離れている(padding: 1000px;)

0pxを指定する場合は、デフォルトの状態で「padding: 0px;」だから記述する必要がない。

marginは風船の「皮」から外側のエリア。どのサイトにもpaddingよりも扱いが難しいと書かれている。
```

#boxモデルという概念と、inline

```
boxモデルは4つの領域で構成されている

1 content（コンテンツ）
    テキストや画像など、要素そのものの内容が表示される領域。
    この領域のサイズは width　と height　プロパティで指定することができる。

2　padding（パディング）
    content と border の間にある余白の領域です。要素と枠線との余白を取るために使います。その余白は padding プロパティで指定することができます。

3　border（ボーダー）
    padding と　margin のあいだに存在する領域、いわゆる枠線。この線の太さは border-width プロパティで指定することができます。

4　margin（マージン）
    ボックスの一番外側の余白の領域です。この領域のサイズは margin プロパティで指定することができます。


インラインブロック要素は、ブロック要素とインライン要素のサラブレッド
display: inline-block;

・改行が入らずに横に並んでいく

・幅と高さが指定できる

・marginとpaddingの値を自由に指定できる

・text-align(中央揃え）と　vertical-align（縦方向の揃え位置を指定する）の指定ができる


```
